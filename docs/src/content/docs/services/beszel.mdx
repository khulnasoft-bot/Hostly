## What is Beszel?

Lightweight server monitoring hub with historical data, docker stats, and alerts.

## Setup

- Deploy Beszel using Hostly template
- In the UI, `Add a new System`
- Enter `beszel-agent` in Host/IP
- Copy the key from the modal in the `KEY` environnement variable in Beszel's docker-compose env section (Edit Docker Compose button of Beszel service in Hostly UI)
